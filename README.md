## 🇺🇸 English

### Project Overview
This project is a high-fidelity Front-end prototype designed for **MushMC**, one of Brazil's largest Minecraft servers. The goal is to create a dedicated hub for community suggestions, solving the noise and organization issues found in traditional forums or Discord channels.

### 🎨 Key UI/UX Features
- **Upvote/Downvote System:** Prioritizes what the community truly wants, making the staff's decision-making easier.
- **Categorization by Tags:** Visual badges for specific modes (BedWars, SkyWars, Server) to improve navigation.
- **Dark Mode Design:** Modern and clean interface consistent with the gaming community's visual identity.
- **Engagement Metrics:** Visible comment counts and vote tallies to encourage participation.

### 🛠️ Tech Stack
- **HTML5:** Semantic structure for better accessibility.
- **CSS3:** Custom styles, Flexbox layout, and hover effects for interactivity.

## 🇧🇷 Português

### Sobre o Projeto
Este projeto é um protótipo de Front-end de alta fidelidade desenvolvido para o **MushMC**. O objetivo é criar um hub centralizado para sugestões da comunidade, resolvendo problemas de organização e poluição visual comuns em fóruns tradicionais ou canais de Discord.

### 🎨 Diferenciais de UI/UX
- **Sistema de Votação (Upvote):** Prioriza o que a comunidade realmente deseja, facilitando a tomada de decisão da Staff.
- **Categorização por Tags:** Badges visuais para modos específicos (BedWars, Servidor, etc.) para melhorar a navegação.
- **Design em Dark Mode:** Interface moderna e limpa, alinhada à identidade visual da comunidade gamer.
- **Métricas de Engajamento:** Contagem visível de comentários e votos para incentivar a participação.

### 🛠️ Tecnologias Utilizadas
- **HTML5:** Estrutura semântica para melhor acessibilidade.
- **CSS3:** Estilização customizada, layout com Flexbox e efeitos de interação (hover).
